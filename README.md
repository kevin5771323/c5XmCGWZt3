## 技术栈 java18

- **前端框架**：Vue3
- **UI库**：Element Plus
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **图表库**：ECharts

## 功能模块

系统包含以下12个页面：

- 登录页
- 注册页
- 仪表盘首页
- 水果分类管理
- 水果库存管理
- 订单管理
- 用户管理
- 角色管理
- 权限管理
- 个人中心
- 系统设置
- 错误页面

此外，还包括以下自定义组件：

- 共15个自定义组件

## 系统特色

- **分页功能**：方便数据浏览与管理。
- **状态管理**：使用Pinia实现状态管理。
- **数据可视化**：集成ECharts进行数据图表展示。
- **组件通信**：支持父子组件通过prop和emit进行通信。
- **指令支持**：包括v-if、v-for、v-on、v-model、v-bind等常用指令。
- **高级特性**：计算属性（computed）与侦听器（watch）。
- **权限控制**：内置RBAC权限控制系统，确保系统安全。

## 系统角色与运行环境

- **系统角色**：面向开发人员，适合学习后台管理系统的开发。
- **运行环境**：任何支持现代Web标准的浏览器。

## 部署步骤

1. 确保本地环境已安装Node.js。
2. 克隆项目到本地。
3. 执行`npm install`或`yarn`安装项目依赖。
4. 运行`npm run dev`或`yarn dev`启动开发服务器。
5. 打开浏览器访问指定端口即可。
